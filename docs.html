SimpleAPI is a basic API using Vercel cloud functions to receive email, source, userAgent, timestamp via POST and store on mongo atlas.

You must suscribe and get your bearer token to use the premium version of the API.

That said, you can still test API and have fun. By default, you are logged in as guest and use the following bearer token token.

As a guest user, you are limiting to 10 calls by minute, but you can tests all the API capabilities: add, delete and export as CSV . For heavier usage, please log in with a premium account.
